# Research Methodology Overview: RQ1 Analytical Framework

This document summarizes the four primary data evaluation streams deployed to track intersectional pathways across educational cohorts.

## 1. Baseline Stratification & Demographics
**Script:** `stratification_analysis.py`
Calculates weighted proportional distributions across structural milestones (Gender, SES, Migration status) ensuring clear oversight of background privileges.

## 2. Foundational Proficiency Mapping
**Script:** `proficiency_analysis.py`
Applies continuous PIAAC cognitive metrics to capture cognitive disparities safely.

## 3. Workplace Skill Applications
**Script:** `workplace_skill_analysis.py`
Examines continuous task complexities.

## 4. Multi-Variant Correlation Matrices
**Script:** `correlation_analysis.py`
Standardizes internal check variations.
